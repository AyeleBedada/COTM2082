Semester Porject

Find the meterials you need for your activity on the fainal project of the course at the button below:

Schematic information of your project

At the button above access the folders named according to your respective names at and find a file named "00_note.txt". This file contains a description of your respective semester project. Based on the description and link (archdaily) in the text file, you will compile a document that contains schematic Plans, Sections, Elevations, and Colored Photographs of your semester project. You can use the compilation at these links as an example: (1) Word file of the example (2) Pdf File of the example.

The file you compiled will be used as a reference (start point) throughout the semester while your work on the:
(1) Construction drafting,
(2) Preparation of a complete set of working drawings and
(3) Coordination among different disciplines in the built environment such as Architectural, Sanitary, Electrical, HVAC, and Mechanical. Furthermore, you will coordinate your drawings documents with schedule and codes, standards and specification documents. Schematic information of your project
